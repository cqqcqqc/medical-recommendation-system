# 医患推荐系统 — 学术研究与前端实践项目

> 基于 Flask 框架开发的智能医患匹配平台，旨在解决医疗信息不对称问题，提升医疗资源匹配效率。本仓库同时收录项目配套学术论文与核心技术实现。

## 📌 项目核心信息
- **项目名称**：医患推荐系统
- **技术栈**：Flask + HTML5 + CSS3 + JavaScript + TailwindCSS + JMeter
- **项目定位**：学术研究 + 前端全栈实践

## ✨ 核心功能模块
- **用户管理**：注册、登录、注销，支持角色权限控制与密码加密存储
- **患者/医生管理**：信息录入、列表展示、编辑与删除，支持专长与症状管理
- **智能推荐**：基于症状-专长匹配算法，实现精准医患推荐
- **套餐与科普**：医疗套餐展示、医学知识科普、系统公告滚动
- **响应式界面**：适配多端访问，支持 50 并发性能测试

## 📊 性能与安全成果
- 并发 50 用户下平均响应时间 < 2s，满足高并发场景需求
- 防 SQL 注入、XSS 攻击，用户信息加密存储，安全防护完善
- 患者找医效率提升 70%，有效优化医疗资源配置
