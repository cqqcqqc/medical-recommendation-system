## 一、项目简介
本项目是一个集**学术研究**与**前端全栈实践**于一体的智能推荐系统。核心目标是通过智能算法，实现患者与医生的精准匹配，优化医疗资源配置，同时为前端开发与系统性能优化提供实践参考。

仓库同步收录了项目配套的**学术论文**与**核心技术实现代码**，适合作为毕业设计、技术研究或前端全栈学习的参考案例。

## 二、核心技术栈
| 分类 | 技术选型 |
| :--- | :--- |
| **后端框架** | Flask |
| **前端技术** | HTML5 + CSS3 + JavaScript + TailwindCSS |
| **性能测试** | JMeter |
| **核心算法** | 基于症状-专长匹配的智能推荐算法 |

## 三、核心功能模块
1. **👤 用户管理**：注册、登录、注销，支持角色权限控制与密码加密存储。
2. **🩺 患者/医生管理**：信息录入、列表展示、编辑与删除，支持专长与症状管理。
3. **🧠 智能推荐**：基于症状-专长匹配算法，实现精准医患推荐。
4. **📚 科普与公告**：医疗套餐展示、医学知识科普、系统公告滚动。
5. **📱 响应式界面**：适配多端访问，支持 50+ 并发性能测试。

## 四、性能与安全成果
- **高性能**：并发 50 户下平均响应时间 < 2s，满足高并发场景需求。
- **高安全**：防御 SQL 注入、XSS 攻击，实现用户信息加密存储，安全防护完善。
- **高收益**：患者找医效率提升 70%，有效优化医疗资源配置。
